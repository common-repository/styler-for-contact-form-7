Styler for Contact Form 7 helps you to create beautiful designs without CSS Coding.

== Description ==


**Style Contact Form 7 without any CSS Coding.** This free plugin comes with 50+ options to design different sections for form like form wrapper, labels, text fields, dropdowns, radio inputs, checkbox inputs , paragraph text area, submit button and many more.

You can customize background color, font size, font family, borders, width, text alignment, margin, padding, font color and more without writing a single line of CSS code. 

Create beautiful themes and templates without hiring a designer. Layout for each form is stored sepreately so that you can create different design for each form.


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/styler-for-contact-form-7` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Now go to any post/page(in frontend) where you have added a form. Make sure you are logged in.
4. Click on the Customize button to launch Wordpress Customizer.
5. Click on "Styler for Contact Form 7" and then select the Form you want to customize.
6. Wait while the plugin saves your form selection and refreshes the page.
7. Now start designing your form


== Frequently Asked Questions ==

= Can I create different styles for different Forms? =

Yes each form design is stored seprately. So, design for each form can be different

= Can I create multiple styles for one form =

As of now you can only create one design for one form. If you want to create multiple designs for single form then first duplicate it in Contact Form 7 and then create a new design for it.
